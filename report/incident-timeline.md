# Incident Timeline

## Initial Reconnaissance
- Threat actor browsed Azure Crest public website
- Researched IT layoffs and ERP changes
- Identified high-value targets

## Initial Access
- Phishing emails sent with healthcare-themed subjects
- Malicious `.docm` files delivered via spoofed domains

## Execution
- Users downloaded documents via browser
- Macro execution led to secondary payload delivery

## Persistence & Discovery
- Database discovery tools executed
- Files compressed for exfiltration

## Impact
- Database files encrypted
- Systems rendered inaccessible
- Hospital operations disrupted
